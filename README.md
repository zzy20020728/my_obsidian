# my_obsidian

秋招备战 & 论文研究知识库 — 基于 [Karpathy LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) 模式构建

## 架构

- `raw/` — 原始资料（论文 PDF、文章、图片，不可修改层）
- `wiki/` — LLM 维护的知识 wiki（编译层）
- `plans/` — 学习计划与进度追踪
- `templates/` — 页面模板
- `AGENTS.md` — Wiki Schema（LLM 操作指南）

## 工具

- **Obsidian** — 浏览、查看知识图谱
- **OpenCode / Claude** — 维护 wiki，摄入论文，回答问题
- **Git** — 版本管理
