# 2026 秋招备战总计划

## 个人背景
- 中科大研二，研究方向：大模型无监督强化学习（URLVR）
- 当前在小厂实习，学习算法
- Java 零基础
- 目标：2026年9月秋招进入大厂
- 每日可用学习时间：3-4小时（实习之外）

## 战略定位

### 核心策略：算法岗为主（70%），应用开发为保底（30%）

**为什么这样分配：**
1. 中科大硕士 + 大模型 RL 方向 → 天然匹配算法岗
2. Java 零基础想 5 个月达到大厂项目水平不现实，但达到面试基础水平可行
3. 算法岗壁垒（论文+科研经历）是你已经在构建的护城河
4. 在 AI 时代，理解大模型原理 + 能做 RLHF/训练 的人远比 Java CRUD 工程师稀缺

### 投递策略
| 岗位类型 | 投递比例 | 目标公司 |
|----------|----------|----------|
| 大模型算法（RL/对齐/训练） | 60% | 字节、阿里、腾讯、百度、华为、商汤、MiniMax、月之暗面、智谱 |
| 大模型应用开发/平台 | 30% | 字节、阿里、美团、京东、快手 |
| 通用后端开发 | 10% | 保底选择 |
